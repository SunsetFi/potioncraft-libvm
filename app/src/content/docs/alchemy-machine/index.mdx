The Alchemy Machine is a device that can be used to create specialist [salts](/salts) to assist in brewing potions. It is found in the basement, and must be repaired and upgraded to access higher tier salts.

The Alchemy Machine functions by combining multiple potions to produce pre-determined output. Initially, only the recipe for [Nigrido](/alchemy-machine-recipes/nigredo) is unlocked. More recipes and Alchemy Machine upgrades can be purchased from the Fellow Alchemist.

## Alchemy Machine Stages

## Initial State

The Alchemy Machine starts the game broken. It can be repaired by speaking to the Fellow Alchemist and purchasing the repair kit.
