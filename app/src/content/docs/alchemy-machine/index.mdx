The Alchemy Machine is a device that can be used to create specialist [salts](/salts) to assist in brewing potions. It functions by combining multiple potions and reagents to produce a pre-determined output based on the recipe used.

The machine can be found in the basement, and must be repaired and upgraded to access higher tier salts.

[All Alchemy Machine Recipes](./alchemy-machine-recipes)

## Alchemy Machine Stages

### Initial State

The Alchemy Machine starts the game broken. It can be repaired by speaking to the Fellow Alchemist and purchasing the repair kit.
